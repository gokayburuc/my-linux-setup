# MY LINUX SETUP 

This is a bash file to setup my-linux with one click

## USAGE 

 - Clone the Setup : 
```bash
git clone https://github.com/gokayburuc/my-linux-setup
```

- cd into the folder

```bash
cd my-linux-setup/
```

- Start `first-setup.sh` file 

```bash
bash first-setup.sh
```
